<h1> Kaurea - Thousand Golden </h1>

<p>Kaurea is a hashing method with multiple solutions and a size support of unlimited¹.</p>
<p></p>

- This method is purely in C, requiring usage of adaptation for other languages through FFI methods, function adaptation and other methods. 

<foot>Notes: _¹ The hashing method has a maximum size of 4,294,967,295 bytes for 32-bit systems and 18,446,744,073,709,551,615 for 64-bit systems._ _²_</foot>
